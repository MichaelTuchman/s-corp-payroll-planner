# Roadmap

## Version 1.0

- Validate the planner against actual July payroll.
- Correct usability, labeling, and formula defects found through real use.
- Finalize commercial styling and instructions.
- Add workbook protection without blocking legitimate user inputs.
- Confirm clean operation in supported versions of Microsoft Excel.

## Later consideration

- Add a Payroll Register worksheet containing copied values only.
- Add a Record Payroll action if a reliable macro-free or clearly supported implementation is available.
- Add historical summaries based on the value-only register.

## Explicitly outside the current scope

- Multiple employees.
- Multi-company support.
- Payroll processing or tax filing.
- Automated state and local tax-rate selection.
- General bookkeeping.
- Bank feeds.
